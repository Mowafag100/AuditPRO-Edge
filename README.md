# 🛡️ AuditPRO: Smart AI Contract Analyzer

AuditPRO is an enterprise-grade AI platform designed to analyze legal contracts, identify risks, and provide actionable recommendations using local LLMs (Ollama/DeepSeek).

## ✨ Key Engineering Features
- **AI Output Structuring:** Guaranteed JSON responses for precise risk scoring.
- **Observability:** Built-in audit trail and latency monitoring.
- **Security:** JWT-based authentication with SHA-256 password hashing.
- **Exportable Analytics:** One-click PDF report generation.
- **CI/CD:** Automated build pipeline via GitHub Actions.

## 🚀 Quick Start
No complex installation required. Just run:
\`\`\`bash
./setup.sh
./run.sh
\`\`\`

## 🛠️ Tech Stack
- **Backend:** FastAPI (Python 3.10)
- **Frontend:** Next.js 14 (Tailwind CSS)
- **AI Engine:** DeepSeek-R1 via Ollama
- **Database:** SQLite
- **DevOps:** GitHub Actions
